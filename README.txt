=== Installation instructions ===

Place the code of the module into the mod/mrproject directory of your Moodle
directory root. That is, the present file should be located at:
mod/mrproject/README.txt

For further installation instructions please see:
    http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

This module is intended for Moodle 3.10 and above.
